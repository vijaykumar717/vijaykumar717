## 📌 Professional Project Areas

> Due to company confidentiality, I do not publish internal source code, datasets, dashboards, or proprietary architecture.  
> The project areas below describe my professional work at a high level using safe and non-confidential information.

### 1. Case Manager AI Assistant

LLM-powered assistant for Financial Risk Management workflows.  
Built to support customer profile analysis, alert reasoning, transaction history review, false positive scoring, prompt-based investigation, and transaction insights.

**Keywords:** GenAI, LLM, RAG, FRM, Transaction Analytics, Customer Analytics, Prompt Engineering, FastAPI, Local LLM

---

### 2. Real-Time Transaction Anomaly Detection

Machine Learning system for large-scale transaction monitoring using supervised and unsupervised learning models.  
Worked with high-volume transaction data, feature engineering, EDA, PySpark pipelines, Random Forest, Isolation Forest, and false alert reduction.

**Keywords:** Machine Learning, Anomaly Detection, PySpark, Spark ML, Fraud Detection, Transaction Monitoring, Big Data, Feature Engineering

---

### 3. Intelligent Document Automation

OCR, NLP, translation, and local LLM-based document intelligence for banking compliance and LEA workflow automation.  
Focused on extracting action items from documents, mail content, account-related requests, and regulatory communication.

**Keywords:** OCR, NLP, Document Intelligence, PaddleOCR, NLLB, Qwen, IPEX-LLM, Quantization, Compliance Automation

---

### 4. GenAI-Powered Text-to-SQL Analytics Dashboard

Natural language to SQL analytics system connected with BI dashboards.  
Built to convert user questions into SQL queries and generate business charts using Metabase.

**Keywords:** Text-to-SQL, SQLCoder, Gemini API, Metabase, Business Intelligence, Data Visualization, SQL, REST APIs

---

### 5. RAG-Based XML Code Generation Assistant

Internal RAG application for generating structured XML rule engine configurations using natural language prompts, internal documentation, examples, templates, and rule patterns.

**Keywords:** RAG, XML Code Generation, Rule Engine, LLM, Embeddings, Vector Search, Prompt Engineering, FastAPI

---

### 6. Dynamic Website RAG Chatbot

Website-based RAG chatbot where users provide a website URL and the system automatically scrapes public visible content, stores embeddings in a vector database, and answers user questions based on retrieved context.

**Keywords:** RAG, Gemini, Weaviate, Web Scraping, Selenium, BeautifulSoup, Vector Database, Embeddings, Cloud Deployment

---

### 7. NLP Intent Classification System

Deep Learning-based multi-class intent classification system for conversational AI workflows using Bi-Directional LSTM, custom dataset creation, data augmentation, tokenization, and text preprocessing.

**Keywords:** NLP, Deep Learning, Bi-Directional LSTM, Intent Classification, Multi-Class Classification, TensorFlow, Keras, Data Augmentation
